Class Instance

A router instance that each component creates.

This is the reference that is exposed and observable by the caller of the component such as:

Example

<script lang="ts">
  let instance = $state<Instance>();
</script>

<Router
 bind:instance
 {routes}
 pre={globalAuthGuardHook}
 post={globalLoggerPostHook}
 notFoundComponent={NotFound} />

<div>
  Current Path: {instance.current.path}
  Navigating: {instance.navigating}
</div>
